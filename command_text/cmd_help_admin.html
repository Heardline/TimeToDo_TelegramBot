Команды администратора бота: • /log — последние 100 строчек лога • /dump — база данных пользователей • /notify_all — рассылка всем пользователям • /reply — для ответов на фидбек Прочие функции администратора: ✨ Оповещения о новых пользователях бота Скрытые команды (но доступны всем): • /users — список всех пользователей • /users_stats — список всех пользователей с их активностью • /commands — список всех комманд с их популярностью • /donate — донаты разработчику (игрушка с нерабочей оплатой) Команды не для всех: • /chai — вызов на чай • /ch — чай-чат
